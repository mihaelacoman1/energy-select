# Energy Select — Technical Expertise in Electrical Installations

**Energy Select** is a comprehensive web project developed to provide a modern digital presence for electrical consultancy and execution services. The platform is live and can be accessed at: [energyselect.ro](https://energyselect.ro)

## Project Concept and Vision
The objective of this project was to digitize the professional expertise of a **Certified Electrician**. The platform was designed to transition from a traditional service model to a structured digital resource, emphasizing safety, technical precision, and energy efficiency.

* **Core Focus:** Providing clear, technical documentation for automation, maintenance, and photovoltaic solutions.
* **Inspiration:** Bridging the gap between field-tested electrical expertise and modern client accessibility requirements.

## Deployment and Production Process
The transition from local development to a live production environment involved the following technical milestones:

1. **Domain Acquisition:** Secured and configured the `energyselect.ro` domain.
2. **Code Optimization:** Refined HTML5 and CSS3 assets, ensuring modularity across specialized pages (Automation, Maintenance, Photovoltaic Systems).
3. **Server Configuration:** Deployed files to a high-performance web server and implemented **SSL Certification** (HTTPS) to ensure data integrity and secure connections.
4. **Version Control:** Managed the development lifecycle via Git and GitHub to maintain a robust deployment pipeline.
5. **DNS Propagation:** Successfully managed DNS records to link the source code with the official domain.

## Technical Stack
* **Frontend Architecture:** Clean HTML5 and CSS3 utilizing CSS Variables (`:root`), Flexbox, and Grid layouts for responsive performance.
* **Typography & UI:** Implemented the *Plus Jakarta Sans* typeface with custom scroll-reveal logic for a professional user experience.
* **SEO Strategy:** Multi-page structure optimized for service-specific indexing (e.g., Industrial Automation, Green Energy).

## Project Structure
* `index.html` - Primary landing page and service overview.
* `electricitate.html` & `automatizari.html` - Technical specifications for electrical and automation services.
* `panouri.html` - Specialized section for renewable energy solutions.
* `experienta.html` - Documented portfolio and professional certifications.
